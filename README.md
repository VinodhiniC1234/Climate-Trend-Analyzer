# 🌍 Climate Trend Analyzer

---

## 📌 Project Overview

The **Climate Trend Analyzer** is a Data Science and Machine Learning project that analyzes historical and simulated climate data to identify patterns, trends, anomalies, and future climate behavior.  

It helps understand how temperature, rainfall, and humidity change over time using data visualization, statistical analysis, anomaly detection, and forecasting techniques.

This project is designed as a **portfolio-ready GitHub project** to demonstrate real-world data analysis and machine learning skills.

---

## 🎯 Objective

The main objective of this project is to:

- Analyze historical climate data patterns
- Identify long-term temperature and rainfall trends
- Detect unusual climate behavior (anomalies)
- Forecast future temperature trends using ML models
- Visualize climate changes in an understandable way
- Build an end-to-end Data Science pipeline

---

## ⭐ Key Features

- 📊 Climate data generation (synthetic simulation)
- 🧹 Data cleaning and preprocessing
- 📈 Trend analysis of temperature, rainfall, humidity
- 🚨 Anomaly detection using Isolation Forest
- 🔮 Temperature forecasting using Linear Regression
- 🌍 3D interactive climate visualization (Plotly)
- 📉 Time-series based climate analysis
- 📦 Fully modular Python project structure

## 🔄 How It Works

1. **Data Generation**
   - Synthetic climate dataset is created with temperature, rainfall, and humidity values.

2. **Data Preprocessing**
   - Missing values handled
   - Date formatting and feature extraction (year, month)

3. **Trend Analysis**
   - Visualizes climate changes over time using graphs

4. **Anomaly Detection**
   - Uses Isolation Forest to detect unusual climate patterns

5. **Forecasting**
   - Predicts future temperature trends using regression model

6. **Visualization**
   - Generates 2D graphs and 3D interactive climate plots
